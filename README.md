# OpenCVSharpImageProcessing
https://github.com/fukuzai/OpenCVSharpImageProcessing

Image Processing such as threshold and labeling for single image using OpenCVSharp.

# Requirements
#### Visual Studio
   2015 or later
   
#### OpenCVSharp
   Get OpenCVSharp v4.0.30319 from NuGet
   
#### .NET Framework 4.6.1
